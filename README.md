# Paper Bond

## Overview
Paper Bond is a real-time paper trading platform where user can simulate stock market exchange using virtual capital. It is risk free platform where user can practice trading strategy and many more.

## features 
- Real Time stock simulation
- virtual capital makes it risk free
- good for early traders to test there algorithm and gain experience
- manage portfolio
- check profit and loss analysis
- real time buy and sell using virtual capital
